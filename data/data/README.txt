This is the data folder, which contains the folder structure used in the code.
Each directory currenty contains a file "empty.txt", which should be deleted, but are needed for uploading to git.

The training data should be put in the folder training_set
The test data should be put in the folder test_set
The track data should be put in the folder track_features

After processing the data, some files chosen at random from training_set_proc should be put in training_set_proc_val to use for validation.
